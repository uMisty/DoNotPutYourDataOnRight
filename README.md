# Don't put your data on right

将Excel中的表单转换为结构化数据  

例如  
Input 1:
|||||  
|--- |--- |--- | ---
|姓名|张三|年龄|18
|性别|男|描述|一个快乐男孩

Input 2:
|||||  
|--- |--- |--- | ---
|姓名|小红|年龄|16
|性别|女|描述|可能这个名字比较适合表明他是个女孩


Output(Json):
```json
[
  {
    "姓名":"张三",
    "年龄":"18",
    "性别":"男",
    "描述":"一个快乐男孩"
  },
  {
    "姓名":"小红",
    "年龄":"16",
    "性别":"女",
    "描述":"可能这个名字比较适合表明他是个女孩"
  }
]
```

Output(csv/excel):
|姓名|年龄|性别|描述|
|--|--|--|--|
|张三|18|男|一个快乐男孩|
|小红|16|女|可能这个名字比较适合表明他是个女孩|
